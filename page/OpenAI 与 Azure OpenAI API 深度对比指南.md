## 核心术语解析
理解API服务需掌握以下关键指标：
- **RPM** (每分钟请求量) - 系统每分钟处理请求的上限
- **TPM** (每分钟Token数) - 每分钟可处理的文本单元总量
- **RPD** (每日请求量) - 每日服务调用次数限制
- **TPD** (每日Token数) - 每日可处理的文本单元总配额
